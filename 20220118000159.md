# FAQ

- NFT を作るのにコーディング必要？
  - Engin に置いては、GUIでできるので不要
  - プロジェクトに統合するにはコーディングが必要
- 無料？
  - GUI mint interface, API, SDK ,開発ツールは完全無料
  - イーサリアムからNFTをmintingするのに、ある程度必要
    - ネットワーク手数料を払うための ethereum
    - NFT 鋳造材料の Engin Coin
  - JumpNet を使うと 僅かな Engin Coin があれば NFT を鋳造できる
